# Como editar um cliente

## Quando usar
Quando dados do cliente mudaram (endereço, telefone, responsável, etc.).

## Passo a passo
1. Acesse **Clientes**.
2. Abra o cliente na lista.
3. Clique em **Editar**.
4. Atualize as informações.
5. Salve.

## O que acontece depois
As novas informações passam a valer para OS futuras e para o histórico do cliente.

## Erros comuns
- Editar o cliente errado (nomes parecidos).
- Apagar informações importantes sem querer.

## Dicas
- Antes de salvar, confirme se é o cliente correto.
- Mantenha endereço e contato sempre atualizados.
- Se o cliente mudou de responsável, atualize o contato principal.
