# Como cadastrar um cliente

## Quando usar
Quando uma nova empresa/pessoa vai receber atendimento e você precisa cadastrá-la no sistema.

## Passo a passo
1. Acesse **Clientes**.
2. Clique em **Novo cliente**.
3. Preencha nome/empresa e dados de contato.
4. Salve.

## O que acontece depois
O cliente passa a aparecer na lista e pode ser selecionado ao criar uma Ordem de Serviço.

## Erros comuns
- Cadastrar cliente com nome incompleto (dificulta busca).
- Esquecer telefone/e-mail (dificulta contato).

## Dicas
- Padronize nomes (ex.: “Loja Centro – ACME”).
- Cadastre endereço completo para o técnico.
- Se houver mais de um contato, registre o principal.
