# Como ver o histórico de OS de um cliente

## Quando usar
Quando você quer consultar atendimentos anteriores (para recorrência, garantia, histórico de problemas).

## Passo a passo
1. Acesse **Clientes**.
2. Abra o cliente desejado.
3. Entre na seção de **Ordens de Serviço** do cliente (histórico).
4. Filtre por período/status (quando disponível).

## O que acontece depois
Você visualiza as OS já criadas para aquele cliente e pode abrir detalhes, anexos e PDFs (conforme permissões).

## Erros comuns
- Não encontrar OS por falta de filtro (período muito curto).
- Usuário sem permissão para ver OS antigas.

## Dicas
- Use o histórico antes de criar uma nova OS (evita duplicidade).
- Verifique anexos e PDFs de OS anteriores para acelerar diagnóstico.
- Para técnicos, histórico ajuda a entender ambiente e padrão do cliente.
