# Como atualizar dados da empresa

## Quando usar
Quando houve mudança de razão social, telefone, endereço ou dados gerais da empresa.

## Passo a passo
1. Acesse **Configurações**.
2. Abra a seção **Empresa**.
3. Atualize os dados necessários.
4. Clique em **Salvar**.

## O que acontece depois
Os dados da empresa ficam atualizados e podem refletir em PDFs e comunicações.

## Erros comuns
- Atualizar dados incorretos (CNPJ/telefone).
- Esquecer de salvar após editar.

## Dicas
- Atualize esses dados antes de emitir/gerar PDFs para clientes.
- Mantenha telefone e e-mail corretos para facilitar contato.
- Se houver mais de uma unidade, padronize o endereço principal.
