# Como cadastrar um técnico

## Quando usar
Quando você precisa criar acesso para um novo técnico atender ordens de serviço.

## Passo a passo
1. Acesse **Técnicos** (ou Usuários, dependendo do menu).
2. Clique em **Novo técnico**.
3. Preencha nome, e-mail e dados solicitados.
4. Defina o perfil como **Técnico**.
5. Salve e compartilhe o acesso com o técnico.

## O que acontece depois
O técnico passa a aparecer para atribuição em OS e poderá acessar o painel conforme permissões.

## Erros comuns
- Cadastrar com e-mail errado (não consegue entrar).
- Criar como perfil errado (gestor/admin) sem querer.

## Dicas
- Use e-mail do próprio técnico (evite e-mails genéricos).
- Explique ao técnico como atualizar status e anexar evidências.
- Se houver limite por plano, acompanhe o número de técnicos cadastrados.
