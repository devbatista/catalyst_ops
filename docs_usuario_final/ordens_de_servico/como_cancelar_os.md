# Como cancelar uma OS

## Quando usar
Quando o atendimento não vai mais acontecer (cliente desistiu, OS duplicada, erro de cadastro, etc.).

## Passo a passo
1. Abra a OS que será cancelada.
2. Clique em **Cancelar**.
3. Confirme o cancelamento.

## O que acontece depois
O status muda para **Cancelada**. A OS não deve mais ser tratada como atendimento ativo.

## Erros comuns
- Cancelar por engano uma OS importante.
- Cancelar sem registrar motivo (pode confundir relatórios/histórico).

## Dicas
- Registre o motivo do cancelamento na descrição/observações.
- Se a OS só precisa mudar data, prefira **Reagendar**.
- Evite manter OS duplicadas: cancele a que estiver errada.
