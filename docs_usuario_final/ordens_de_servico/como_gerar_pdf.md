# Como gerar PDF da OS

## Quando usar
Quando você precisa exportar a OS para enviar ao cliente, imprimir ou arquivar.

## Passo a passo
1. Abra a OS.
2. Clique em **Gerar PDF**.
3. Aguarde o arquivo ser gerado.
4. Clique para **baixar** ou **visualizar** o PDF.

## O que acontece depois
O sistema gera um PDF com os dados da OS (cliente, descrição, itens, anexos — conforme configuração).

## Erros comuns
- PDF não gera (instabilidade momentânea).
- Dados incompletos no PDF por falta de preenchimento.

## Dicas
- Antes de gerar, revise a OS e itens de serviço.
- Gere o PDF após concluir/finalizar para ter o “fechamento” do atendimento.
- Se o PDF estiver errado, revise campos e gere novamente.
