# Como tratar uma OS atrasada

## Quando usar
Quando uma OS estava agendada e passou do horário, ficando marcada como **Atrasada**.

## Passo a passo
1. Acesse **Ordens de Serviço**.
2. Abra a OS marcada como **Atrasada**.
3. Verifique o motivo (cliente, técnico, falta de peça etc.).
4. Use o fluxo **Reagendar** e defina uma nova data/hora.
5. Salve e confirme o reagendamento.

## O que acontece depois
Ao reagendar, a OS volta a ficar **Agendada** com a nova data/hora e aparece novamente na agenda.

## Erros comuns
- Deixar OS atrasada acumulando (bagunça a operação).
- Reagendar sem avisar cliente e técnico.

## Dicas
- Crie um hábito diário: tratar OS atrasadas no começo do dia.
- Registre o motivo do atraso na descrição para histórico.
- Evite agendar sem folga entre atendimentos.
