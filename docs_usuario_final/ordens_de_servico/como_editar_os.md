# Como editar uma Ordem de Serviço

## Quando usar
Quando você precisa corrigir ou complementar informações (descrição, cliente, dados do atendimento) antes de finalizar.

## Passo a passo
1. Acesse **Ordens de Serviço**.
2. Abra a OS desejada.
3. Clique em **Editar** (ou opção equivalente na tela).
4. Atualize os campos necessários (título, descrição, cliente, técnico, observações).
5. Clique em **Salvar**.

## O que acontece depois
As alterações ficam registradas na OS e passam a valer para a agenda, relatórios e para o técnico (quando aplicável).

## Erros comuns
- Tentar editar uma OS já **Finalizada** (pode estar bloqueada).
- Alterar dados críticos sem avisar o técnico responsável.

## Dicas
- Antes do atendimento, revise a descrição para evitar dúvidas em campo.
- Se mudar cliente/técnico, confirme se a agenda precisa ser ajustada.
- Padronize descrições (ex.: sempre incluir endereço e contato).
