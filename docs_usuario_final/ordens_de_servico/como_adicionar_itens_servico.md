# Como adicionar itens de serviço

## Quando usar
Quando você precisa registrar serviços executados, peças, materiais ou itens que compõem o atendimento.

## Passo a passo
1. Abra a OS.
2. Acesse a seção **Itens de serviço** (ou equivalente).
3. Clique em **Adicionar item**.
4. Preencha descrição/quantidade/valor (se disponível).
5. Salve o item.

## O que acontece depois
Os itens ficam registrados na OS e podem aparecer em PDF/relatórios (dependendo da configuração).

## Erros comuns
- Adicionar item sem descrição clara.
- Registrar quantidade/valor incorretos.

## Dicas
- Padronize nomes de itens para facilitar relatórios.
- Registre materiais usados ainda durante o atendimento para não esquecer.
- Se houver cobrança, revise itens antes de finalizar a OS.
