# Como visualizar relatórios de OS

## Quando usar
Quando você quer analisar quantidade de atendimentos, status, períodos e desempenho operacional.

## Passo a passo
1. Acesse **Relatórios**.
2. Selecione o relatório de **Ordens de Serviço**.
3. Filtre por período (e por cliente/status, se disponível).
4. Analise os resultados na tela.

## O que acontece depois
Os relatórios ajudam a identificar volume de OS, pendências, atrasos e comportamento por período.

## Erros comuns
- Relatório com poucos dados por filtro muito restrito.
- Comparar períodos diferentes sem padronizar o filtro.

## Dicas
- Use períodos fixos (semana/mês) para comparar corretamente.
- Acompanhe OS atrasadas e sem técnico como prioridade operacional.
- Use relatórios para definir melhorias no processo.
