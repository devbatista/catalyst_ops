# Como consultar artigos da base de conhecimento

## Quando usar
Quando você tiver dúvida operacional e quiser orientação rápida.

## Passo a passo
1. Acesse **Base de conhecimento**.
2. Abra a categoria do tema.
3. Leia o artigo e aplique no atendimento.

## Dica
Se o procedimento não resolver, reporte ao gestor com prints e passos executados.
