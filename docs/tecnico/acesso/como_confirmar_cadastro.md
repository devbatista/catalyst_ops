# Como confirmar cadastro

## Quando usar
Quando você foi cadastrado e recebeu o e-mail para **definir/redefinir senha**.

## Passo a passo
1. Abra o e-mail de redefinição de senha.
2. Clique no link recebido.
3. Defina sua nova senha e confirme.
4. Salve e faça login normalmente.

## Resultado
Ao concluir a redefinição, sua conta fica ativa para uso do sistema.
