# Como ver minhas Ordens de Serviço

## Quando usar
Use este fluxo quando quiser consultar as OS atribuídas a você, verificar detalhes do atendimento ou abrir uma OS para executar o serviço.

## Passo a passo
1. Acesse **Ordens de Serviço** no menu lateral.
2. Veja a lista das OS atribuídas ao seu usuário.
3. Use os filtros disponíveis, se necessário.
4. Clique na OS desejada para abrir os detalhes.

## O que acontece depois
Você consegue ver informações como cliente, descrição do serviço, status, datas, anexos e observações da OS.

## Erros comuns
- Procurar uma OS que ainda não foi atribuída ao seu usuário.
- Abrir a OS errada sem conferir cliente, código ou descrição.

## Dicas
- Priorize OS com status **Agendada**, **Em andamento** ou **Atrasada**.
- Sempre confira a descrição antes de sair para o atendimento.
- Use o código da OS para localizar mais rápido um atendimento específico.
