# Como ver detalhes da OS

## Quando usar
Antes de iniciar atendimento, para confirmar dados do cliente e escopo do serviço.

## O que revisar
- Título e descrição.
- Cliente e endereço.
- Status atual.
- Observações anteriores.
- Anexos já enviados.

## Dica
Revise sempre os detalhes antes de alterar o status.
