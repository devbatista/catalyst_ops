# Como iniciar um atendimento

## Quando usar
Use este fluxo quando você estiver começando a execução do serviço e quiser registrar no sistema que o atendimento foi iniciado.

## Passo a passo
1. Abra a OS atribuída a você.
2. Revise cliente, descrição e horário do atendimento.
3. Clique em **Iniciar**.
4. Confirme a ação.

## O que acontece depois
O status da OS muda para **Em andamento**. Isso indica que o atendimento já começou.

## Erros comuns
- Tentar iniciar uma OS cancelada ou já finalizada.
- Iniciar a OS errada sem validar o cliente.

## Dicas
- Antes de iniciar, confira se você está no local certo e na OS correta.
- Se necessário, adicione anexos logo no começo do atendimento.
- Se houver impedimento para executar o serviço, fale com o gestor antes de mudar o status.
