# Como atualizar status da OS (regras do técnico)

## Status que o técnico consegue usar
- **Iniciar**: muda para `Em andamento`.
- **Concluir**: muda para `Concluída`.

## Restrições do perfil técnico
- Não pode mudar para `Finalizada`.
- Não pode mudar para `Cancelada`.
- Não pode reagendar OS atrasada.
- Na edição da OS, técnico só pode alterar **observações** e **anexos**.

## Dica
Se precisar cancelar/finalizar/reagendar OS atrasada, peça ação do gestor.
