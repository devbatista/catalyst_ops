# Como usar o Dashboard do técnico

## O que você encontra
- Agenda do dia.
- OS em andamento.
- OS atrasadas.
- OS concluídas/finalizadas no mês.

## Como usar no dia a dia
1. Abra o Dashboard ao iniciar o turno.
2. Confira a agenda e priorize os próximos atendimentos.
3. Acompanhe OS em andamento para não perder contexto.
4. Avise o gestor se houver impedimento ou atraso.

## Dica prática
Mantenha o Dashboard aberto para acompanhar mudanças de status durante o dia.
