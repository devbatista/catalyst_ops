# Como aceitar o contrato de utilização

## Quando usar
Quando o sistema exigir aceite do contrato para liberar o acesso.

## Passo a passo
1. Abra a tela de **Contrato de utilização**.
2. Leia o conteúdo.
3. Marque a opção de aceite.
4. Clique em **Aceitar**.

## Dica
Sem o aceite, o acesso operacional pode ficar bloqueado.
