# Como atualizar meu perfil

## Quando usar
Use este fluxo quando precisar corrigir seus dados pessoais, trocar senha ou manter seu cadastro atualizado.

## Passo a passo
1. Acesse **Configuração** no menu lateral.
2. Abra a seção do seu perfil.
3. Atualize os campos necessários, como nome, email, telefone ou senha.
4. Clique em **Salvar**.

## O que acontece depois
Se os dados estiverem válidos, o sistema salva as alterações e seu perfil passa a refletir as informações atualizadas.

## Erros comuns
- Informar email inválido.
- Alterar senha e esquecer de confirmar corretamente.
- Salvar sem revisar os dados digitados.

## Dicas
- Mantenha telefone e email atualizados para facilitar contato da empresa.
- Use uma senha forte e fácil de lembrar.
- Se uma alteração não funcionar, revise os campos obrigatórios antes de tentar de novo.
