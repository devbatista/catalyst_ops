# Como redefinir senha

## Quando usar
Quando você já está logado e precisa trocar sua senha.

## Passo a passo
1. Acesse **Configurações**.
2. Abra a aba **Perfil**.
3. Preencha os campos **Nova senha** e **Confirme a nova senha**.
4. Clique em **Salvar alterações**.

## Resultado
A senha da conta é atualizada no seu perfil.
