# Documentação Para Agentes De IA

Esta pasta concentra contexto de desenvolvimento para agentes de IA. Ela não
substitui `README.md`, `README_TECNICO.md` nem a central de ajuda em `docs/`;
ela aponta quais regras de negócio e riscos devem ser considerados antes de
alterar código.

## Como Usar

- Comece por `AGENTS.md` na raiz do projeto.
- Leia o arquivo de domínio relacionado à tarefa.
- Quando uma tarefa cruzar domínios, leia todos os arquivos envolvidos.
- Para fluxos de usuário, consulte também `docs/gestor/` e `docs/tecnico/`.
