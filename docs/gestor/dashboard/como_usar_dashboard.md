# Como usar o Dashboard

## O que você encontra
- Indicadores de clientes, técnicos e ordens de serviço.
- Distribuição de OS por status.
- Cards com variação semanal.
- Listas rápidas de OS recentes e pendentes de aprovação.

## Como usar no dia a dia
1. Abra o Dashboard no início do expediente.
2. Verifique OS em atraso, em andamento e concluídas.
3. Priorize pendências e distribua técnicos.
4. Acompanhe variação semanal para identificar gargalos.

## Dica prática
Use o Dashboard como painel de triagem e execute ações nas telas de OS, agenda e técnicos.
