# CatalystOps — Central de Ajuda (Padrão SaaS por Fluxo)

Esta central foi escrita para **empresas (gestores)** e **técnicos**, em **PT‑BR**, com linguagem **simples → intermediária**.

## Como usar esta ajuda
Procure pelo objetivo que você quer atingir, por exemplo:
- “Como agendar um atendimento”
- “Como gerar PDF da OS”
- “Como anexar fotos na OS”
