# Como ver os detalhes de uma Ordem de Serviço

## O que aparece na tela de detalhe
- Dados do cliente e descrição.
- Status atual e horários.
- Técnicos atribuídos.
- Itens de serviço.
- Anexos e observações.

## Quando usar
Sempre que você precisar validar contexto antes de alterar status, reagendar, anexar arquivos ou gerar PDF.
