# Como remover um anexo da OS

## Quando usar
Quando um arquivo foi anexado por engano ou precisa ser substituído.

## Passo a passo
1. Abra a OS.
2. Vá até a área **Anexos**.
3. Localize o arquivo que deseja remover.
4. Clique em **Remover/Excluir** (quando disponível).
5. Confirme.

## O que acontece depois
O anexo é removido da OS e deixa de aparecer para os usuários.

## Erros comuns
- Remover o arquivo errado (sem backup).
- Remover evidências importantes antes de finalizar.

## Dicas
- Antes de remover, confirme se não é exigência do seu processo (fotos, laudos).
- Se for substituir, anexe o novo arquivo logo em seguida.
- Se não aparecer a opção de remover, fale com o gestor/admin.
