# Como criar uma Ordem de Serviço

## Quando usar
Use este fluxo sempre que um cliente solicitar um atendimento técnico e você precisar registrar o serviço no sistema.

## Passo a passo
1. Acesse **Ordens de Serviço** no painel (**app**).
2. Clique em **Nova OS**.
3. Preencha **Título** e **Descrição** (o que será feito).
4. Selecione o **Cliente**.
5. Se já souber quem irá atender, selecione o **Técnico** (quando aplicável).
6. Clique em **Salvar**.

## O que acontece depois
A OS é criada e aparece na listagem. Em geral, ela começa como **Pendente** até você **agendar** (data/hora) e/ou **iniciar** o atendimento.

## Erros comuns
- Não selecionar um cliente.
- Deixar título/descrição vazios ou muito genéricos.
- Criar OS sem informações suficientes para o técnico.

## Dicas
- Use títulos claros (ex.: “Instalação de câmera – Loja Centro”).
- Na descrição, informe sintomas, endereço e observações importantes.
- Se o atendimento depende de peças, registre isso na OS para evitar retorno.
