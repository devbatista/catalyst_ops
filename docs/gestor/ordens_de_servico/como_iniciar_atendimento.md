# Como iniciar um atendimento

## Quando usar
Quando o técnico começa o atendimento no local do cliente e você quer registrar o início da execução.

## Passo a passo
1. Abra a OS do dia (pela lista ou pela agenda).
2. Clique em **Iniciar**.
3. Confirme a ação.

## O que acontece depois
O status muda para **Em andamento**. A OS passa a ser tratada como atendimento em execução.

## Erros comuns
- Tentar iniciar uma OS cancelada/finalizada.
- Iniciar a OS errada (cliente diferente).

## Dicas
- Antes de iniciar, confirme endereço e descrição do serviço.
- Se o atendimento não vai ocorrer, reagende ou cancele em vez de iniciar.
- Use anexos (fotos) para registrar evidências do início, quando necessário.
