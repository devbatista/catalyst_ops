# Como ver a lista e filtrar Ordens de Serviço

## Quando usar
Quando você precisa localizar uma OS específica por código ou status.

## Passo a passo
1. Acesse **Ordens de Serviço**.
2. Use o filtro por **status**.
3. Use a busca por **código**.
4. Abra a OS desejada para editar, atualizar status ou agendar.

## Dicas
- Trate primeiro OS com status **Atrasada** e **Sem técnico**.
- Use o código da OS para localizar mais rápido.
