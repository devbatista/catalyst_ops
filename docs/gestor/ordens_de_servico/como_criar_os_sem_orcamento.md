# Para criar Ordens de Serviço sem Orçamentos

## Quando usar
Quando sua operação precisa abrir OS diretamente, sem depender do fluxo de orçamento.

## Pré-requisito
A regra **Criar OS sem orçamento** deve estar ativa em **Configurações**.

## Passo a passo
1. Acesse **Configurações**.
2. Na seção **Dados da Empresa**, clique em **Editar dados**
3. Ative **Criar OS sem orçamento**.
4. Clique em **Salvar**.
5. Volte para **Ordens de Serviço**.
6. Clique em **Nova OS** e preencha os dados obrigatórios.

## O que acontece depois
- A OS é criada sem vínculo com orçamento.
- O fluxo operacional segue normalmente (agendar, iniciar, concluir/finalizar, cancelar).

## Erros comuns
- Tentar criar OS direta com a regra desativada.
- Não preencher campos mínimos obrigatórios da OS.

## Dicas
- Use essa opção para chamados urgentes.
- Defina internamente quando usar OS sem orçamento para manter padrão de processo.
