# Como remover um técnico

## Quando usar
Quando o técnico não fará mais parte da operação.

## Passo a passo
1. Acesse **Técnicos**.
2. Abra o registro do técnico.
3. Clique em **Excluir**.
4. Confirme a ação.

## Atenção
Se houver vínculos que impeçam exclusão, o sistema exibirá mensagem de validação.
