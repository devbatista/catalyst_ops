# Como editar um técnico

## Quando usar
Quando precisar atualizar nome, e-mail ou telefone de um técnico.

## Passo a passo
1. Acesse **Técnicos**.
2. Abra o técnico desejado.
3. Clique em **Editar**.
4. Atualize os dados e salve.

## Dica
Mantenha e-mail e telefone corretos para comunicação operacional.
