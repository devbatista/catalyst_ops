# Como atualizar seu perfil

## Quando usar
Quando você precisa alterar dados do seu usuário (nome, informações de contato, etc.).

## Passo a passo
1. Acesse **Configurações**.
2. Abra a seção **Perfil**.
3. Atualize os dados necessários.
4. Clique em **Salvar**.

## O que acontece depois
As informações do seu usuário ficam atualizadas para o sistema e para comunicações internas.

## Erros comuns
- Salvar dados incompletos.
- Trocar e-mail sem saber se impacta o login (depende do seu fluxo).

## Dicas
- Use dados reais para facilitar contato do suporte.
- Se trocar e-mail, confirme se o login utiliza esse e-mail.
- Mantenha o nome completo para evitar confusão em atribuições.
