# Como aceitar o contrato de utilização

## Quando usar
Quando o sistema solicitar aceite do contrato antes de continuar.

## Passo a passo
1. Abra a tela de **Contrato de utilização**.
2. Leia o conteúdo.
3. Marque a opção de aceite.
4. Clique em **Aceitar**.

## O que acontece depois
O aceite é registrado com usuário, data/hora e contexto de acesso.
