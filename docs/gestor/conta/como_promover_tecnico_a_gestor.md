# Como promover técnico para gestor

## Pré-requisitos
- Você deve ser o responsável da empresa para realizar promoção.
- O usuário selecionado deve estar como técnico.
- O plano não pode ser o plano básico.

## Passo a passo
1. Acesse **Configurações** > **Gestores**.
2. Em "Promover técnico a gestor", selecione o técnico.
3. Clique em **Promover**.

## O que acontece depois
O usuário passa a ter permissões de gestor.
