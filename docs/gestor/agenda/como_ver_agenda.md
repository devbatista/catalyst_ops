# Como ver a agenda

## Quando usar
Quando você quer visualizar atendimentos por dia/semana para organizar a operação.

## Passo a passo
1. Acesse **Agenda** no painel.
2. Escolha o dia/semana que deseja visualizar.
3. Clique em um item para abrir a OS (quando aplicável).

## O que acontece depois
Você vê os atendimentos agendados e consegue abrir detalhes para confirmar horário, cliente e descrição do serviço.

## Erros comuns
- Não ver atendimentos porque a OS não está agendada.
- Ver agenda vazia por filtro ativo (técnico/período).

## Dicas
- Use a agenda no começo do dia para planejar rotas.
- Ao reagendar uma OS, confira se a agenda atualizou.
- Para técnicos, mantenha a agenda aberta durante o dia.
