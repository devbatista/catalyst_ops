# Como enviar orçamento para aprovação

## Quando usar
Quando o orçamento já está revisado e pronto para o cliente decidir.

## Passo a passo
1. Abra o orçamento em **Rascunho**.
2. Clique em **Enviar ao cliente**.
3. O sistema envia e-mail com link de aprovação/rejeição para o cliente.

## O que acontece depois
- O orçamento muda para **Enviado**.
- O cliente pode abrir o link, baixar o PDF e aprovar/rejeitar.
- O link expira na data de validade do orçamento.

## Erros comuns
- Tentar enviar orçamento sem itens.
- Informar e-mail inválido no cadastro do cliente.
