# Como aprovar ou rejeitar orçamento (gestor)

## Quando usar
Quando for necessário decisão interna do gestor (o ideal é aprovação do cliente).

## Passo a passo
1. Acesse **Orçamentos**.
2. Abra o orçamento.
3. Para aprovar, clique em **Aprovar (Gestor)**.
4. Para rejeitar, clique em **Rejeitar (Gestor)** e informe motivo.

## O que acontece depois
- **Aprovar**: orçamento vai para **Aprovado** e o sistema cria a OS automaticamente com status **Pendente**.
- **Rejeitar**: orçamento vai para **Rejeitado** com motivo.

## Dicas
- Registre motivo de rejeição claro para orientar a revisão.
- Priorize decisão do cliente sempre que possível.
