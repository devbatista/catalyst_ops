# Como criar um orçamento

## Quando usar
Quando você precisa iniciar um novo atendimento no fluxo atual (a OS será gerada após aprovação do orçamento).

## Passo a passo
1. Acesse **Orçamentos** no menu lateral.
2. Clique em **Novo Orçamento**.
3. Selecione o cliente.
4. Preencha título, descrição e validade.
5. Adicione os itens de serviço (descrição, quantidade e valor unitário).
6. Salve.

## O que acontece depois
O orçamento é salvo como **Rascunho** e fica pronto para envio de aprovação.

## Dicas
- Cadastre itens completos para o total refletir corretamente.
- Use descrição objetiva para facilitar aprovação do cliente.
