# Como visualizar a área Financeira

## Quando usar
Use esta área quando quiser acompanhar o valor dos itens lançados nas Ordens de Serviço, separando o que já foi **realizado** do que ainda está **pendente**.

## Passo a passo
1. Acesse **Financeira** no menu lateral.
2. Veja os cards de resumo no topo da tela.
3. Confira:
   - **Faturamento Realizado**: valor das OS **finalizadas**.
   - **Faturamento Pendente**: valor das OS ainda não **finalizadas**.
   - **Volume Total**: soma de todos os itens lançados nas OS consideradas.
4. Use o filtro da tabela para escolher entre:
   - **Todas**
   - **Realizado**
   - **Pendente**
5. Analise a lista de OS para ver o detalhe por cliente, status e valor.

## O que acontece depois
Você consegue ter uma visão financeira operacional do que já foi entregue e do que ainda está em aberto, usando os itens lançados nas OS como base.

## Erros comuns
- Ler **pendente** como valor já recebido.
- Esquecer que o cálculo depende dos **itens de serviço** cadastrados na OS.
- Esperar que OS **canceladas** apareçam nessa visão.

## Dicas
- Finalize as OS assim que o processo terminar para manter o realizado atualizado.
- Revise OS com status **concluída** e **atrasada**, pois elas impactam o valor pendente.
- Use esta tela em conjunto com a listagem de OS para investigar diferenças de valor.
