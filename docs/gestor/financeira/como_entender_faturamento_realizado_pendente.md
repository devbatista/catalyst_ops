# Como entender faturamento realizado e pendente

## Quando usar
Use este guia quando quiser interpretar corretamente os números mostrados na área **Financeira**.

## Passo a passo
1. Abra a área **Financeira**.
2. Observe o status das Ordens de Serviço.
3. Entenda a separação:
   - **Realizado**: OS com status **Finalizada**.
   - **Pendente**: OS com status **Pendente**, **Agendada**, **Em andamento**, **Concluída** ou **Atrasada**.
4. Verifique o valor de cada OS na tabela.
5. Compare os totais dos cards com os registros listados abaixo.

## O que acontece depois
Você passa a usar a tela com mais segurança para acompanhar o valor operacional das OS e identificar onde ainda existe trabalho ou fechamento pendente.

## Erros comuns
- Confundir **Concluída** com **Finalizada**.
- Considerar o valor pendente como faturamento fechado.
- Cadastrar itens sem quantidade correta, o que altera o total financeiro da OS.

## Dicas
- Sempre confira **quantidade** e **valor unitário** dos itens lançados.
- Use **Finalizada** como referência para o valor já consolidado.
- Se o valor parecer baixo, revise se a OS recebeu todos os itens de serviço necessários.
