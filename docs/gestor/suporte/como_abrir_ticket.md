# Como abrir um ticket de suporte

## Quando usar
Quando a base de conhecimento não resolver ou houver falha no sistema.

## Passo a passo
1. Acesse **Suporte** > **Tickets**.
2. Clique em **Abrir novo ticket**.
3. Preencha assunto, descrição, categoria, impacto e prioridade.
4. (Opcional) Relacione uma OS.
5. (Opcional) Anexe arquivos.
6. Salve.

## Dica
Tickets com contexto claro (passo a passo + print + erro) são resolvidos mais rápido.
