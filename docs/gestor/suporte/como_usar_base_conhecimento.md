# Como usar a Base de Conhecimento

## Quando usar
Quando quiser resolver dúvidas sem abrir ticket.

## Passo a passo
1. Acesse **Suporte** > **Base de conhecimento**.
2. Procure artigos por categoria e título.
3. Aplique o passo a passo do artigo no seu fluxo.

## Dica
Abra ticket somente quando não houver artigo aplicável ou quando o problema persistir.
