# Problemas comuns (Troubleshooting)

## Não consigo criar uma OS
- Verifique se você está com perfil **gestor/admin**
- Confirme se o **cliente** está cadastrado
- Preencha **título** e **descrição** com informações mínimas

## Não consigo agendar
- Data/hora não pode estar no passado
- Confira se a previsão de término está preenchida (se obrigatório)

## A OS ficou “Atrasada”
- Reagende a OS para uma nova data/hora
- Confirme com o cliente e com o técnico

## Não consigo anexar arquivos
- Tente um arquivo menor
- Tente outro formato
- Verifique sua conexão

## Técnico não aparece para atribuição
- Confirme se o usuário está cadastrado como **Técnico**
- Verifique se ele está ativo e com acesso liberado

## Não consigo entrar no sistema
- Confirme e-mail e senha
- Use “Recuperar senha”
- Se persistir, peça ao gestor/admin para verificar seu usuário
