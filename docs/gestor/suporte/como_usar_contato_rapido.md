# Como usar o Contato Rápido

## Quando usar
Quando você precisa de um canal direto e seu plano possui esse recurso.

## Passo a passo
1. Acesse **Suporte** > **Contato rápido**.
2. Verifique se o recurso está habilitado no seu plano.
3. Use o canal indicado para acionar o time.

## Observação
O Contato rápido depende do plano contratado.
