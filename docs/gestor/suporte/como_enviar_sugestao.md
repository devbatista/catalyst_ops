# Como enviar sugestão de melhoria

## Quando usar
Quando quiser sugerir evolução de produto, processo ou usabilidade.

## Passo a passo
1. Acesse **Suporte** > **Sugestões**.
2. Preencha título, tipo, impacto e descrição.
3. Envie a sugestão.

## Dica
Descreva cenário real de uso e ganho esperado para priorização correta.
